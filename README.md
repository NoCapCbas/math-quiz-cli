# Math Quiz CLI

## Usage

Clone Repo & Navigate to root directory
```shell
git clone ... & cd ...
```

Build Executable
```shell
go build .
```

Run Executable
```shell
./math-quiz-cli
```

Example
```shell
bash-3.2$ ./math-quiz-cli
Problem #1: 1+0 = 
1
Problem #2: 1+1 = 
3
You scored 1 out of 2.
```
